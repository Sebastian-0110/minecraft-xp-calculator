A client-side Minecraft XP calculator built with Vue.js that allows users to calculate the number of actions needed to reach specific XP levels. This tool helps players optimize their gameplay by providing insights into XP requirements for various in-game actions.

## Features

- Calculate required actions for leveling up based on different XP sources
- User-friendly interface built with Vue.js
- Supports various actions like killing mobs and mining resources
- Real-time updates as users input their desired XP level

## Technologies Used

- **Frontend**: Vue.js
- **JavaScript**: For interactive calculations
- **HTML/CSS**: For structuring and styling the application
